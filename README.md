# Angular + Bootstrap Skeleton

Este es un esqueleto de Angular listo para **copiar/pegar** bloques de Bootstrap y CSS.
- Bootstrap 5 por CDN (no necesitas instalar nada extra para estilos).
- Componentes standalone listos: Login, Personas, Insumos, Citas, Pagos, Empleados, Reportes, Dashboard.
- Archivos `.html` con marcadores `START-PASTE` para que pegues tu código Bootstrap.

## Uso
1. `npm install`
2. `npm start`
3. Abre `http://localhost:4200`

## Dónde pegar tu HTML/CSS
- Login: `src/app/auth/login.component.html`
- Personas: `src/app/persons/persons.component.html`
- (y así con cada módulo)

Puedes añadir CSS en `src/styles.css`.
